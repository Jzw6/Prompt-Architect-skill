# Prompt Architect

**A comprehensive prompt engineering skill for AI assistants.**

Prompt Architect transforms vague ideas into production-ready prompts through a structured, multi-phase workflow:

1. **Requirement Clarification** — A nine-dimension questioning framework that dynamically adapts to the user's context, avoiding unnecessary interrogation.
2. **Architecture Design** — Visual-first approach with Mermaid diagrams to map out prompt structure before writing a single word.
3. **Prompt Writing** — Structured, platform-aware output following battle-tested templates and anti-hallucination patterns.
4. **Evaluation & Iteration** — A five-dimension scoring rubric to objectively assess and improve any prompt.

## Key Features

- 🎯 **Platform-Adaptive** — Tailored guidance for GPT, Claude, Gemini, open-source models, agent frameworks, and GPTs/custom bots.
- 🧩 **Modular Templates** — Reusable building blocks for system prompts, chain-of-thought, few-shot examples, and guardrails.
- 🔄 **Iterative by Design** — Built-in feedback loops that refine prompts through testing and edge-case analysis.
- 🌐 **Multilingual Ready** — Handles mixed-language input, colloquial phrasing, and non-standard formatting gracefully.
- 📊 **Visual Architecture** — Mermaid-based flowcharts to communicate complex prompt logic at a glance.

## When to Use

- "I want an AI that does X" → Full prompt design from scratch
- "Improve / debug this prompt" → Diagnosis + targeted rewrites
- "Evaluate this prompt" → Structured scoring with actionable quick wins
- Building multi-step AI pipelines or agent workflows
